Brainf*ck can be hard to code, but it doesn't have to be! Here are some tips to make coding it a bit easier.



The Basics

  • > and < move the pointer within a tape. Try opening the console with Ctrl+Shift+I on Chromium browsers, and type >>><<< into the brainf*ck shell
  • + and - increment and decrement within a tape. Open the console and type --+++++. Notice how the value wraps around
  • . and , output and input characters respectively. Try doing ,+.+.+. and input a character. It should output the next three characters
  • [ ] create loops. The code only progresses when the current tape value is 0. Open the console and see what this program does. +++++[>+++<-],
  • For more info about loops, check out more below


General Tips

  • Create loops with []. Use a cell as an index, and another as the working cell:
    1. Create a index: >
    2. Set index size, for example, 3: >+++
    3. Start loop and move to working cell: >+++[>
    4. Do your action, for example, add 4: >+++[>++++
    5. Move back to index, decrement, and close loop >+++[>++++<-]

  • Factor your index sizes. For example, to type A, use three for loops with 3, 4, 5, and add 5 to the final index to get there faster
    • Example code: +++[>++++[>+++++<-]<-]>>+++++.

  • Use an ASCII chart. There is one at the bottom of the page
  • Brainf*ck is confusing enough. Use comments with " " to make your life easier, and use spaces.
    • Example code: "1st index" +++ [ >++++"2nd index" [ >+++++ "Start dec." <-]<-] >> +++++ "Print".

  • Stick to the default environment variables when starting out. If you know what you are doing, modify the variables.
